# mip-huimee

mip-huimee 第三方广告投放插件

标题|内容
----|----
类型|广告
支持布局|responsive,fixed-height,fill,container,fixed
所需脚本|http://mipcache.bdstatic.com/static/v1/mip-huimee/mip-huimee.js

## 示例

### 基本用法
```html
<mip-fixed  type="bottom" id="huimee">
	<mip-huimee> </mip-huimee>
</mip-fixed>
```

### mip-huimee

说明：第三方广告投放组件
