# mip-audio 音频播放

提供了一个音频播放组件

描述|内容
----|----
类型|通用
支持布局| N/S
所需脚本|https://mipcache.bdstatic.com/static/v1/mip-audio/mip-audio.js

## 示例

```
<mip-audio 
	src="http://yinyueshiting.baidu.com/data2/music/122873158/49046814400128.mp3?xcode=1b2a4f4315be4ae75990d376263109e0"
></mip-audio>
	
```

## 属性

### src

说明：音频地址
必填：是
格式：url
使用限制：必须是https的
